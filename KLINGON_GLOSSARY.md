## Translation Notes

1. **Compound Terms**: Many ONI concepts don't exist in Klingon, so compound words are created using existing vocabulary.

2. **Technical Precision**: Where Klingon lacks specific technical terms, descriptive phrases are used (e.g., "power maker device" for generator).

3. **Cultural Adaptation**: Some concepts are adapted to fit Klingon cultural context while maintaining game meaning.

4. **Consistency**: Related terms use consistent root words to maintain logical relationships.
